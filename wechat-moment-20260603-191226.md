Yoda v0.3.7 已发布。相比 v0.3.6，这版主要解决多 agent 编码里几个很现实的问题：上下文看不清、会话不好恢复、退出应用怕打断正在跑的任务。

我最在意的是 Context 面板。之前 Claude 会话能看得比较清楚，但 Codex 里模型拿到了哪些运行时信息、memory、tools 和 prompts，不够直观。这版开始支持 Codex；Claude/Codex 的 skills、agents 也会从用户目录、项目目录、插件目录实时扫描，不再只看启动时快照。

tmux 体验也补上了。退出 Yoda 前，如果还有 agent session 在跑，会提示哪些可以留在 tmux 继续跑，哪些会被停止；tmux 不可用时也会给安装入口。长期任务不再靠记忆兜底。

日常使用也更顺：任务右键可复制 session id、项目路径和 resume 命令；无项目任务进入 Drafts 统一管理；标题点击即可重命名；Home 支持 Typeless 语音输入。底层同步升级到 Electron 41.7.1，并做了一轮稳定性修复。
